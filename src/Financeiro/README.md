# Controle de Gastos Pessoais

Este projeto é um sistema simples em Java para controle de gastos pessoais, com persistência dos dados em arquivo CSV.

## Funcionalidades

- Cadastro de novos gastos (descrição, categoria, valor, data)
- Edição e remoção de gastos existentes
- Consulta de gastos cadastrados
- Persistência dos dados em arquivo `gastos.csv`
- Interface via console

## Pré-requisitos

- Java 17 ou superior instalado
- IntelliJ IDEA (ou outra IDE Java)
- Git (opcional, para clonar o repositório)
